About

This is a simple sign-in-out application designed for Sorrento Surf Life Saving Club.

It allows members to easily log their attendance for activities, starting with Youth Sunday Sessions.
The intention is to provide an extremely simple electronic replacement of the traditional paper sign-in sheets.

If this application is useful, it may be possible to extend it's use to Nippers or other activities.
Please speak with Mick O'Connor if you would like to discuss this further.

How the app works

Currently, the sign-in-out forms can only be used on Sunday mornings, during standard Youth/Nipper session times.
i.e.
Sign in is accessible from 8:00am to 9.30am
Sign out is accessible from 9:31am to 10:40am.

Youth simply enter their name and click the "Sign In/Out" button.
Only registed Youth will be able to sign in. Unrecognised names will be rejected.

Age Managers have addiitonal features to see live number of sign-ins, historical data, etc.
Again, these features are only available during standard Youth/Nipper session times.

FAQ

I am a registered memember, but my name is not listed (or is incorrect)

Please use paper sign-in backup, and discuss with Age Managers, so missing/incorrect name can be corrected.

I would like some extra features added to the app - e.g. tracking profeciencies/awards, etc

This is not the purpose of the app... but if used successfully for sign-in-out, then perhaps this could be considered in future.

I am an Age Manager and I incorrectly signed someone in/out - how can I delete/correct this?

Currently, there is no way to delete data... please be more careful :D
The ability to delete/edit data will be added in the future.

The site currently says the next session is this weekend, but the season is over!

The sign-in-out application is not aware of the season dates... it simply assumes that sessions are every Sunday.
Please ignore the "Next Session" message when the season is over and during xmas break :)
